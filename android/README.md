For building using cmake and vcpkg on Windows:

```
.\vcpkg.exe install sqlitecpp:arm64-android
.\vcpkg.exe install sqlitecpp:x86-android
.\vcpkg.exe install sqlitecpp:x64-android
.\vcpkg.exe install protobuf:arm64-android
.\vcpkg.exe install protobuf:x86-android
.\vcpkg.exe install protobuf:x64-android
.\vcpkg.exe install gtest:arm64-android
.\vcpkg.exe install gtest:x86-android
.\vcpkg.exe install gtest:x64-android
```
